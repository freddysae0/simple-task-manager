
# Simple Task Manager (Backend, Frontend, CLI):


## Backend:

### How to run:

```bash
pnpm install
pnpm run dev
```

### API Endpoints:

GET /api/v1/tasks - Get all tasks
GET /api/v1/tasks/:id - Get task by ID
POST /api/v1/tasks - Create new task
PUT /api/v1/tasks/:id - Update task
DELETE /api/v1/tasks/:id - Delete task
GET /api/v1/done - Get completed tasks
GET /api/v1/pending - Get pending tasks
